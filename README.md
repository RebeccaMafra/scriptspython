## Meus Scripts

Este repositório contém os scripts que desenvolvi enquanto pratico programação.

## Scripts incluídos

- **calculadorateste.py**  
  Este script é uma calculadora básica onde estou aprendendo a usar funções, entrada de dados e operações matemáticas.

- **atividade1.py** 
    Este script é um saudação,onde utilizei funções como 'def', além de utilizar () para definir parametros

- **atividade3.py** 
    script contem comprimento e despedidas, feitos com diferentes funções 

- **atividade5.py** 
    Esse script é feito no objetivo de fazer uma calculadora básica, utilizando as mesmas funções das atividade1 e atividade3

- **calculadora2.py**
    Esse script é uma calculadora infinita, utilizando loop,e a função 'match case'

- **semana3.py** 
    script de um verificador de idade, onde pergunta o usuário a sua idade e seu nome, onde devolve o usuario se ele é ou não maior de idade.
